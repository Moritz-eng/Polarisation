Polarisation Visualization

Dieses Repository enthält ein Python-Skript zur Visualisierung von Polarisation. 
Es zeigt lineare vertikale Polarisation auf der linken Seite und zirkuläre Polarisation auf der rechten Seite,
jeweils mit Pfeilachsen und diskreten E-Feld-Vektoren.
Das Skript verwendet Python mit NumPy und Matplotlib und wurde mit Unterstützung von KI erstellt.
